## 為什麼選擇 n8n？

[n8n](https://n8n.partnerlinks.io/h1pwwf5m4toe) 是一個開源的工作流自動化平臺，讓你可以將任何服務連接到一起。與閉源替代方案不同，n8n 讓你完全掌控自己的資料和基礎設施。核心優勢包括：

- **開源且可自託管** -- 在你自己的伺服器上運行，無供應商鎖定
- **400+ 內置集成** -- 幾乎可以連接任何服務或 API
- **視覺化工作流編輯器** -- 通過拖放節點構建自動化，無需編碼
- **AI 原生能力** -- 內置支援 OpenAI、Claude、Gemini、LangChain 和向量資料庫
- **免費起步** -- n8n Cloud 提供慷慨的免費額度，也可零成本自託管

無論你是在自動化郵件工作流、構建 AI 聊天機器人、處理文檔，還是編排 DevOps 流水線，n8n 都能為這些範本提供運行基礎。

### 如何從本倉庫導入 n8n 範本？

下載你想使用的範本的 `.json` 文件。打開你的 n8n 實例（自託管或 [n8n Cloud](https://n8n.partnerlinks.io/h1pwwf5m4toe) 均可），導航到 Workflows，點擊 "Import from File"，然後選擇下載的 JSON 文件。工作流將出現在你的編輯器中，可以直接配置使用。啟動工作流前，你需要為每個連接的服務添加自己的憑據。

### 什麼是 n8n？為什麼應該用它來做自動化？

n8n 是一個免費的開源工作流自動化平臺，內置超過 400 種集成。與 Zapier 或 Make 等 SaaS 替代方案不同，n8n 可以自託管在你自己的基礎設施上，讓你完全掌控資料。它具有視覺化拖放編輯器、原生 AI 和大語言模型支援，以及活躍的社區。你可以在 n8n 的雲平臺上[免費開始使用 n8n](https://n8n.partnerlinks.io/h1pwwf5m4toe)，也可以部署到你自己的伺服器上。

### 這些範本是免費使用的嗎？

是的，本倉庫中的所有範本完全免費下載和使用。它們是從互聯網上公開可用的來源收集而來，在此分享以便於訪問。n8n 本身是開源的，可以免費自託管。n8n Cloud 平臺也提供額度充裕的免費層級，因此你可以零成本開始使用。

### 我可以貢獻自己的範本嗎？

當然可以。我們非常歡迎貢獻。如果你構建了一個其他人可能會覺得有用的 n8n 工作流，請提交一個 Pull Request，將你的範本 JSON 檔放在相應的類別資料夾中，並附上清晰的標題和描述。你也可以通過在本倉庫中開一個 Issue 來建議新類別或提出改進意見。

### 這些範本支援哪些 AI 模型？

這些範本集成了多種 AI 模型和服務提供者，包括 OpenAI GPT-4 和 GPT-4o、Anthropic Claude、Google Gemini 和 Vertex AI、DeepSeek R1、Mistral AI、LangChain、Perplexity AI、NeurochainAI 以及 Hugging Face 模型。許多範本使用 Pinecone、Qdrant、Supabase 和 Elasticsearch 等向量資料庫來構建 RAG（檢索增強生成）管道。

### 本倉庫多久更新一次？

本倉庫持續積極維護，隨著 n8n 社區提供新範本而定期更新。新類別和新範本會不斷添加。你可以 Watch 或 Star 本倉庫以獲取新增內容的通知。查看頁面頂部的最後提交徽章可以瞭解最近的更新日期。
